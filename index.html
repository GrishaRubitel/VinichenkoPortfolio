Виниченко Даниил

  1. Учебный проект “Сетевой интерфейс для видеокоммутатора Ross”

    Описание: Сервис по “перехвату” информации, передаваемой видеокоммутатором Ross. Одной из функций сервиса является звуковой аналог Tally Lights - озвучка номера камеры, которая переходит в режим Program (картинка с неё идёт в эфир)

    Обязанности:

    • Тимлид
    • Бэкенд Python-разработчик

    Ссылка на Github: https://github.com/GenryEden/RossServer

  1. Учебный проект “Веб-дашборд для VMix и видеокоммутатора Ross”

    Описание: Дашборд собирает информацию об эфирной ситуации с программного микшера VMix (конкурент OBS'а) и коммутатора Ross (см. предыдущий проект). Вывод информации производится через веб-интерфейс.

    Обязанности:

    • Фронтенд-разработчик
    • Бэкенд Python-разработчик

    Ссылка на Github: https://github.com/GrishaRubitel/DashboardTeleS

  1. Учебный проект “Русскоязычный аналог веб-игры Semantle”

    Описание: Простая словесная веб-игра, где нужно угадать слово, основываясь на её семантику. Анализ смыслов искомого и заданного пользователем слов реализован при помощи word2vec

    Обязанности:

    • Фронтенд-разработчик
    • Бэкенд Golang-разработчик

    Ссылки на Github:

    1. https://github.com/GrishaRubitel/Semantle (Фронтенд)
    1. https://github.com/GrishaRubitel/SemantleBack (Бэкенд)

    Ссылка на сайт (ссылка может быть неактивна): http://semantle.ru/

  1. Учебный проект “Nexign Bootcamp”

    Описание: Упрощенный прототип биллинговой системы расчёта звонков с приминением микросервисной архитектуры.

    Ссылки на Github:

    1. https://github.com/GrishaRubitel/BootcampShowcase (Версия для Docker, расписана README)
